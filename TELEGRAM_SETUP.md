# Telegram Integration Setup Guide

## Overview
This guide helps you set up real-time Telegram notifications for trading signals.

## Step 1: Create a Telegram Bot

1. Open Telegram and search for **@BotFather**
2. Send the command `/newbot`
3. Follow the prompts:
   - Choose a name for your bot (e.g., "Crypto Signal Bot")
   - Choose a username (must end with "bot", e.g., "my_crypto_signal_bot")
4. **Copy the API token** provided - you'll need this!

Example token format: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`

## Step 2: Create a Telegram Group (or use existing)

1. Create a new group in Telegram
2. Add your bot to the group (search for your bot's username)
3. Give the bot admin/send message permissions

## Step 3: Get Your Group Chat ID

### Method A: Using curl (Recommended)
1. Send a message in the group (any message)
2. Run this command in your terminal:
   ```bash
   curl "https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates"
   ```
   Replace `<YOUR_BOT_TOKEN>` with your actual token
3. Look for the `"chat":{"id":-1001234567890}` value
4. Copy the negative ID (including the "-")

### Method B: Manual check
1. Send a message in the group
2. Forward it to [@userinfobot](https://t.me/userinfobot)
3. The bot will show you the group ID

## Step 4: Configure Environment Variables

### Linux/Mac
```bash
export TELEGRAM_BOT_TOKEN="your_bot_token_here"
export TELEGRAM_CHAT_ID="your_chat_id_here"
```

### Windows (Command Prompt)
```cmd
set TELEGRAM_BOT_TOKEN=your_bot_token_here
set TELEGRAM_CHAT_ID=your_chat_id_here
```

### Windows (PowerShell)
```powershell
$env:TELEGRAM_BOT_TOKEN="your_bot_token_here"
$env:TELEGRAM_CHAT_ID="your_chat_id_here"
```

### Using .env file (Optional)
Create a `.env` file in the project root:
```
TELEGRAM_BOT_TOKEN=your_bot_token_here
TELEGRAM_CHAT_ID=your_chat_id_here
```

The app now loads `.env` automatically via `python-dotenv`, so no extra step is required before running. If you prefer, you can still use:
```bash
source .env  # On Linux/Mac
```

## Step 5: Run the Application

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Signals will now be sent to your Telegram group automatically! 🚀

## Step 6: Test the Integration

Call the signal endpoint:
```bash
curl http://localhost:8000/live-signal/BTCUSDT
```

You should see a message in your Telegram group like:
```
🚀 Signal for BTCUSDT

Action: BUY
Prediction: 0.123456
Market Regime: Uptrend

💡 Generated by Crypto Signal Engine
```

## Docker Setup

If using Docker, pass environment variables:

### Docker run
```bash
docker run -p 8000:8000 \
  -e TELEGRAM_BOT_TOKEN="your_token" \
  -e TELEGRAM_CHAT_ID="your_chat_id" \
  crypto-signal-engine
```

### Docker Compose
Update `docker-compose.yml`:
```yaml
environment:
  - TELEGRAM_BOT_TOKEN=your_bot_token_here
  - TELEGRAM_CHAT_ID=your_chat_id_here
```

## Troubleshooting

### Bot not sending messages?
- ✅ Bot is in the group and has send message permissions
- ✅ Chat ID is correct (should start with `-100` for groups)
- ✅ Token is valid and hasn't been revoked
- ✅ Check logs: `logger.info` messages will show if messages were sent

### "Telegram notifier disabled" in logs?
- Set both `TELEGRAM_BOT_TOKEN` and `TELEGRAM_CHAT_ID` environment variables
- Reload the application after setting variables
- Check that the values don't have extra spaces or quotes

### Getting "Forbidden" error?
- Bot needs to be admin in the group
- Or give bot explicit permission to send messages
- In Telegram group settings → Members → find bot → change permissions
